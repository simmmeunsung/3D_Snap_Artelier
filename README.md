## Getting Started

First, run the development server:

- 프로젝트 파일에서 bash 실행 후 명령해줍니다:
  
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

만약 Next.js 미설치돼있을시
```bash
npm install next react react-dom

```
해준뒤 위 명령어 해줍니다

그리고나서 사이트를 방문해보세요 ! 

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
